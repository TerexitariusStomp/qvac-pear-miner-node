One-Line Integration

Add distributed AI inference mining to any app or website with a single script tag.

Quick Start

Paste this script into your HTML, WebView, or APK layout:

<script
  src="./inference-embed.js"
  data-app-id="my-awesome-app"
  data-evm-address="0x..."
  auto-install>
</script>

How It Works

1 Auto-Detect

The script reads your device RAM, CPU cores, battery, GPU, and storage.

2 Register

Uses your EVM address to register with the mining coordinator.

3 Earn

When the device is idle, it polls for inference tasks and processes them.

Configuration Attributes

AttributeRequiredDescription
srcYesURL to inference-embed.js
data-app-idYesYour app identifier for attribution
data-evm-addressYesYour EVM wallet address (0x...)
data-backend-urlNoDefault: http://localhost:3000
data-poll-intervalNoTask poll interval in ms. Default: 10000
data-max-concurrentNoMax parallel tasks. Default: 1
auto-installNoAuto-start mining on page load
data-debugNoEnable console logging

Android APK Integration

In your WebView or layout XML, load a local HTML file containing the script:

<!-- assets/embedded-miner.html -->
<!DOCTYPE html>
<html>
<head>
  <meta name="viewport" content="width=device-width, initial-scale=1">
</head>
<body>
  <script src="file:///android_asset/inference-embed.js"
    data-app-id="com.example.myapp"
    data-evm-address="0x..."
    data-backend-url="http://10.0.2.2:3000"
    auto-install>
  </script>
</body>
</html>

JavaScript API

After the script loads, a global PearEmbedMiner object is available:

PearEmbedMiner.status()      // { running, registered, idle, earnings, ... }
PearEmbedMiner.start()       // Begin mining
PearEmbedMiner.stop()        // Stop mining
PearEmbedMiner.on('taskComplete', (data) => console.log(data))
PearEmbedMiner.on('registered', () => console.log('Miner registered'))
